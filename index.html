MaFIN

A concept using Matrices as Finite State Automata

Try the example

The assessment of Complex Problem Solving requires specific tests and assessment approaches. Aiming to minimalize the differences between Complex Problem Solving assessment and Problem Solving asssessment, we introduce the MaFIN approach.

Example item

Explanation

The mafin property of each JSON-object description allows for each cell to define and add an unlimited number of figures and figure properties.

JSON code:

{
  "id": "a",
  "type": "cell",
  "template": "elcol.cell",
  "mafin": {
    "type": "input",
    "influences": {"d": ["size", "rotation"]},
    "figures": [{
      "figure": "full square",
      "size": ["small", "small", "huge"],
      "position": 2,
      "rotation": 45,
      "color": ["blue", "black"],
      "eigendynamics": null
    }, {
      "figure": "circle",
      "size": ["small", "small", "huge"],
      "position": 7,
      "rotation": 45,
      "color": "black",
      "eigendynamics": null
    }]
  }
}
